# 🚀 Smart Code Snippet Finder

A Python-based intelligent code search engine built using Flask and TF-IDF ranking algorithm.

## 🔥 Features
- Keyword-based search
- TF-IDF ranking algorithm
- Modern responsive UI
- Syntax highlighting
- Copy to clipboard
- Real-time search

## 🛠️ Tech Stack
- Python
- Flask
- HTML, CSS
- JavaScript
- TF-IDF Algorithm

## ▶️ How to Run

1. Install dependencies:pip install flask
2. Run application:python app.py
3. Open browser:http://127.0.0.1:10000/
